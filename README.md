Tabela interativa da Copa do Mundo Fifa 2026

Funciona como uma tabela de preenchimento dos resultados de forma manual pelo usuário.
Funcionalidades:
- Atualiza classificação, adotando os critérios de desempate da Fifa
- Calcula os melhores 3º lugares de acordo com os critérios Fifa
- Possui indicação visual de classificados/eliminados
- Salva os resultados preenchidos em local storage
- Backup em arquivo json
- Restaura back
- limpa resultados preenchidos
- modo escuro
- Responsividade aprimorada
