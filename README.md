# 🏆 Tabela Copa do Mundo 2026

Aplicação web desenvolvida em HTML, CSS e JavaScript para simular e acompanhar a Copa do Mundo FIFA 2026.

O projeto permite preencher resultados de todas as partidas da competição, acompanhar a classificação dos grupos em tempo real, avançar automaticamente para as fases eliminatórias e visualizar o campeão ao final do torneio.

A aplicação foi projetada para funcionar tanto em computadores quanto em dispositivos móveis, podendo ser instalada como um aplicativo (PWA).

---

## ✨ Funcionalidades

### Fase de Grupos

* Preenchimento dos resultados de todas as partidas.
* Classificação atualizada automaticamente.
* Critérios de desempate aplicados em tempo real.
* Destaque visual para equipes classificadas e eliminadas.
* Suporte à classificação dos melhores terceiros colocados conforme o regulamento da Copa de 2026.

### Mata-Mata

* 16 avos de final
* Oitavas de final.
* Quartas de final.
* Semifinais.
* Disputa de terceiro lugar.
* Final.

Os confrontos são gerados a partir dos critérios de chaveamento da Fifa.

### Persistência de Dados

* Salvamento automático dos resultados no navegador.
* Manutenção dos resultados mesmo após fechar a aplicação.
* Possibilidade de editar resultados já confirmados.

### Resumo Final

* Tela especial para exibição do campeão.
* Destaque para a seleção vencedora e sua bandeira.
* Exibição dos países que avançaram às fases finais.

### Experiência do Usuário

* Interface responsiva para desktop, tablet e celular.
* Navegação entre rodadas.
* Destaque para partidas do dia.
* Tema visual otimizado para leitura e preenchimento rápido.
* Aplicação instalável como PWA (Progressive Web App).

## Instalação

A aplicação pode ser utilizada diretamente pelo navegador ou instalada como aplicativo em dispositivos compatíveis.

### Desktop

1. Abra o site.
2. Clique no ícone de instalação do navegador.
3. Confirme a instalação.

### Android

1. Abra o site no navegador.
2. Selecione "Instalar aplicativo" ou "Adicionar à tela inicial".
3. Confirme a instalação.

## Objetivo do Projeto

Este projeto foi desenvolvido como um desafio pessoal de programação, com foco em:

* Manipulação de dados em JavaScript.
* Desenvolvimento de interfaces responsivas.
* Lógica de classificação esportiva.
* Armazenamento local de dados.
* Criação de aplicações web instaláveis (PWA).

Além de servir como ferramenta para acompanhar a Copa do Mundo 2026, o projeto também representa uma oportunidade de aprendizado e evolução contínua no desenvolvimento web.
